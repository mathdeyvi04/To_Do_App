# Descrição

Visando organizar minhas ideias momentâneas e armazená-las por um tempo
indeterminado, eu criei esse App capaz de guardá-las para mim.

# Como usar

O executável, criado utilizando Pyinstaller, é capaz de ser usado diversas
vezes.

Há uma barra de inserir texto, no caso a tarefa a ser guardada, com ENTER,
ela é adicionada.

Clicando duas vezes na tarefa listada, ela desaparece.

# Possíveis Melhorias

* Adição de novas ferramentas, como edição de tarefas;
* Retirar necessidade de destruir a tela e reconstruí-la sempre que houver
adição ou subtração de tarefas.
